Bem-vindo ao Distopia Nerd!

Você sabe o que é uma Realidade Distópica?

Você lembra do filme Blade Runner?! A distopia é a representação de uma realidade ou sociedade imaginária, muitas vezes opressora e aterrorizante. É o oposto de utopia. O termo distopia acaba por caracterizar um gênero artístico que apresenta um imaginário sobre o futuro.
